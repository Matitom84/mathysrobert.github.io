Présentation

Le projet

Une PME m'a confié la mission de créer un système capable de répondre automatiquement aux questions de ses clients via WhatsApp. L'objectif : libérer l'équipe des réponses répétitives et offrir une disponibilité 24h/24, 7j/7.

J'ai conçu et déployé un agent IA complet sur n8n, connecté à l'API Meta WhatsApp Business. Le système reçoit les messages des clients, les analyse, consulte une base de données Google Sheets contenant les informations de l'entreprise, puis génère une réponse personnalisée via OpenAI et la renvoie automatiquement.

Le projet m'a pris 2 mois de développement, entre la configuration des APIs, la conception du workflow n8n, la gestion de la mémoire conversationnelle et les tests en conditions réelles.

Comment ça fonctionne

Architecture du système

Un client envoie un message WhatsApp → le WhatsApp Trigger reçoit le message en temps réel via l'API Meta
Un Switch analyse le type de message — entraînement ou question client — et route vers le bon agent
L'agent IA consulte Google Sheets pour récupérer les données de l'entreprise et génère une réponse contextuelle
La mémoire conversationnelle permet à l'agent de se souvenir des échanges précédents avec chaque client
La réponse est renvoyée automatiquement via WhatsApp Business Cloud en quelques secondes
